# Styleguide-Dokumentation

Diese Dokumente ergänzen den Komponentenbaukasten in `index.html`.

- `ai-development-guide.md`: Architektur, Ziele und Regeln für die Weiterentwicklung durch KI oder Menschen.
- `migration-lumitronix-design-to-global.md`: Übersetzung alter `lumitronix-design.css` Muster auf die neue `global.css`.
- `workitem-logik-review-v2.html`: Beispiel einer statischen Dokumentseite, die im Wesentlichen aus dem neuen Komponentenbaukasten aufgebaut ist.

Für neue Komponenten gilt: zuerst `ai-development-guide.md` lesen, dann `index.html` und `global.css` gemeinsam ändern, danach die Migrationstabelle aktualisieren, falls alte Klassen betroffen sind.
